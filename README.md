# dotfiles

Set up procedure for myself.

## Installation

1. Set up **dotfiles**

   ```sh
   $ cd ~
   # First execution of `git` command, git is automatically installed.
   $ git clone https://github.com/ohbarye/dotfiles.git
   $ sh ~/dotfiles/setup.sh
   ```

1. Install **Apps** with Brewfile

   ```sh
   $ cd ~/dotfiles
   $ brew bundle
   ```
